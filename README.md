# shiny

A tool for the crystal programming language for aesthetic and functional TUIs

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     shiny:
       github: NitosMaster/shiny
   ```

2. Run `shards install`
